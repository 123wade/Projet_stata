# Analyse de la pauvreté au Sénégal

Ce projet analyse la pauvreté au Sénégal à partir des données EHCVM_Welfare_SEN2021 (ANSD).

## Objectifs
- Calcul des indices FGT (P0, P1, P2)
- Analyse selon région, sexe, milieu et CSP

## Données
- Source : ANSD
- Année : 2021

## Méthode
- Indices de Foster-Greer-Thorbecke
- Pondération par poids ménages
NB : pour une vue plus claire des resultats, excecuter le code et et regarder le fichier excel
